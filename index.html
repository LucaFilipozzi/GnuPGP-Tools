GnuPGP-Tools

some tools for GnuPGP (GPG)

I'm writing some tools for GnuPGP (GPG) to make live simpler for both: beginners and advanced user of encryption with Gnu's free PGP implementation.

gpgkeymgr

With gpgkeymgr you can clean up and manage your GnuPGP-keyring, by removing old and unnecessary keys.

Features:

  • remove expired keys
  • remove revoked keys
  • remove not-valid keys, optional with given level
  • remove not-trusted keys, optional with given level
  • remove key listed in a file
  • backup you keyring
  • AND or OR mode for key choosing
  • secret key will not be touched
  • internationalization: German and French translations included

Download

  • Download version 0.3 (tar.gz, 37 KB), sha256, sig
  • or get latest version from Github

Dependencies

GnuPGP, gpgme

Examples

Remove all keys wich are expired or revoked: 

$ gpgkeymgr -o -e -r

Remove all keys wich wich are not valid and I don't trust: 

$ gpgkeymgr -v -t

First backup keyring, then delete all keys witch are not completely valid: 

$ gpgkeymgr -b -v 3
